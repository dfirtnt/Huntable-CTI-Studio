How to read this table ▼

Cell layout

  • Large number — actual observable count extracted by the agent
  • Signed delta — actual − expected (e.g. +3 means over-extracted by 3)
  • Expected column — ground-truth count from the evaluation dataset

Badge colors

  • ✓ Exact — delta = 0
  • ±1 Close — |delta| ≤ 2
  • ±3+ Miss — |delta| > 2
  • pending — not yet completed
  • failed — extraction error
  • ⚠ — TPM rate limit hit; raise Concurrency Throttle and re-run
  • ctx↗ — Context window exceeded; article too long for model
  • no mdl — LMStudio not ready when run; re-run to get a real result

MAE chart metrics

  • MAE — avg(|actual−expected|) in observable units; lower is better
  • Green node — full eval set completed (every scheduled article reached completed)
  • Amber node — degraded run: subset of eval articles selected, OR records failed/pending, OR any run was rate-limited or quota-exceeded (those keep completed status but the MAE is unreliable). Hover for the breakdown.
